# 🔁 Проверка строки на палиндром

Современный учебный мини-проект на Python с графическим интерфейсом для проверки, является ли данная строка палиндромом.

## ✨ О проекте

Это аккуратное и наглядное приложение, которое помогает понять, как работает классический алгоритм проверки палиндрома. Пользователь вводит слово, фразу или предложение, а программа автоматически:

- приводит текст к нижнему регистру;
- удаляет пробелы, знаки препинания и лишние символы;
- сравнивает символы слева и справа;
- показывает результат в понятном и красивом виде.

## 🚀 Сильные стороны проекта

- Классический учебный алгоритм через два указателя.
- Понятная структура проекта с разделением логики и интерфейса.
- Красивый графический интерфейс с заголовками и иконками.
- Адаптивное поведение при изменении размера окна.
- Подходит для начинающих.
- Не требует установки сторонних библиотек.
- Идеален для учебной работы, портфолио и демонстрации базовых навыков Python.

## 🧠 Что умеет программа

- Проверяет, является ли строка палиндромом.
- Игнорирует пробелы.
- Игнорирует знаки препинания.
- Игнорирует разницу между большими и маленькими буквами.
- Показывает очищенную строку.
- Показывает понятный результат.
- Позволяет быстро очистить данные и начать новую проверку.

## ▶️ Как запустить

-> Откройте проект (например, в PyCharm).
-> Запустите файл main.py.
-> Введите строку в поле ввода.
-> Нажмите кнопку Проверить.
-> Посмотрите результат и очищенную строку.
